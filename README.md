A modern, responsive, and accessible landing page for a fictional **Vintage Record Store**, built using **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project demonstrates clean frontend development practices, responsive UI design, accessibility, and basic client-side functionality without relying on any external frameworks.
